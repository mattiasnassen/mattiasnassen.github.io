A Pen created at CodePen.io. You can find this one at http://codepen.io/fullkornslimpa/pen/jxiHd.

 NES-gamepad built with css3. No images. No javascript apart from the d-pad (which is a canvas). Try pressing the buttons.

Supported in modern webkit or gecko-based browsers and IE9+

The fonts used are Pretendo (http://jackster3000.deviantart.com/art/Pretendo-Version-2-7242224) and NES-Controller http://mrshrike.deviantart.com/art/NES-Console-Controller-Font-141383845.
